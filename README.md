# Tesiomed
# Tesiomed is a android application which is used for the educational website "www.tesiomed.com". The Android application allows you to Register the users and watch the educational videos which are integrated to youtube.
# The project is useful to the students who are in the medical field and can refer these videos for better knowledge.
# The users need to install the Builded apk file and need to register in the application.
# The application stores the users in cloud and if users are logged in successfully then user is allowed to respective videos.
